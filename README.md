## 🌟 نظرة عامة | Overview

**المنصة التعليمية الذكية** هي تطبيق ويب تقدمي (PWA) متكامل يتيح للطلاب والمعلمين إنشاء وإدارة بنوك أسئلة تفاعلية، واللعب بأوضاع متعددة، ومشاركة البيانات عبر الشبكة المحلية — كل ذلك **بدون الحاجة لخادم خلفي أو اتصال إنترنت** بعد التحميل الأول.

تم تطوير المشروع بأحدث تقنيات الويب الأمامية (Frontend) مع التركيز على:
- ⚡ **السرعة**: تحميل فوري وتشغيل بدون انتظار
- 📴 **العمل Offline**: قاعدة بيانات محلية عبر IndexedDB
- 📱 **التوافقية**: يعمل على جميع الأجهزة (موبايل، تابلت، كمبيوتر)
- 🌙 **الوصولية**: دعم الوضع الداكن/الفاتح واللغتين العربية والإنجليزية

> **الإصدار الحالي:** `v3.0.0` | **تاريخ البناء:** `2025-06-02`

## ✨ المميزات الرئيسية | Key Features

### 🎮 أوضاع اللعب التفاعلية
| الوضع | الوصف | المميزات |
|-------|-------|----------|
| ⚡ **التحدي السريع** | إجابات فورية مع مؤقت تنازلي | مؤقت لكل سؤال، إيقاف مؤقت محدود، نجوم حسب السرعة |
| 📝 **اختبر معلوماتك** | اختبار شامل بمدة زمنية | مدة قابلة للتخصيص، تصحيح تلقائي، مراجعة الإجابات |
| 📊 **الإحصائيات** | تتبع الأداء التفصيلي | نسبة النجاح، أطول سلسلة، النجوم المجمعة |

### 🗄️ إدارة المحتوى
- ✅ **إضافة أسئلة** يدوياً مع دعم الصور
- 📂 **تصنيفات** مرنة (مواد، وحدات، دروس)
- 📤 **تصدير** أسئلة التصنيفات بصيغة JSON
- 📥 **استيراد** من ملفات نصية (.txt) أو JSON (.json)
- 📋 **برومبت ذكي** جاهز للاستخدام مع نماذج الذكاء الاصطناعي (ChatGPT, Claude)

### 🎨 الواجهة والتخصيص
- 🌙 **12+ سمة لونية** (أزرق ذهبي، زمردي، ياقوتي، كوني، نيون...)
- ☀️ **الوضع الفاتح** والداكن مع فصل تام بين الألوان
- 🌐 **دعم اللغتين** العربية (RTL) والإنجليزية (LTR)
- 📱 **تصميم متجاوب** يتكيف مع جميع أحجام الشاشات
- 🎊 **تأثيرات احتفالية** (Confetti) عند النجاح

### 🌐 الشبكة والمشاركة (WebRTC)
- 🔗 **اتصال P2P** مباشر بين الأجهزة عبر WebRTC
- 👨‍🏫 **وضع الأستاذ**: إنشاء غرفة ومشاركة الأسئلة
- 👨‍🎓 **وضع الطالب**: الانضمام عبر مسح QR
- 📱 **مشاركة QR**: لنقل بنك الأسئلة بين الأجهزة
- 🔒 **نظام أذونات**: التحكم بصلاحيات الطلاب المتصلين

### ⚙️ الأنظمة الذكية
- 🧠 **مدقق أسئلة ذكي** يتحقق من صحة البيانات المستوردة
- 🏆 **نظام الإنجازات** والمكافآت
- ⭐ **نظام XP** (نقاط الخبرة) مع 6 مستويات (مبتدئ → أسطورة)
- 🔊 **نطق صوتي** للأسئلة (Text-to-Speech)
- 🎵 **مؤثرات صوتية** تفاعلية (صح/خطأ/انتهاء الوقت)

## ⚡ التثبيت والتشغيل | Installation

### المتطلبات
- متصفح حديث يدعم **IndexedDB** و **ES6+**
- (اختياري) خادم ويب محلي لتشغيل PWA بشكل كامل

### 🚀 التشغيل السريع
```bash
# 1. استنساخ المستودع
git clone https://github.com/username/smart-learning-platform.git

# 2. الدخول للمجلد
cd smart-learning-platform

# 3. تشغيل خادم محلي (مثال باستخدام Python)
python -m http.server 8000
# أو باستخدام Node.js
npx serve .

# 4. فتح التطبيق
# افتح المتصفح على: http://localhost:8000
```

### 📱 التثبيت كتطبيق (PWA)
1. افتح التطبيق في المتصفح (Chrome/Edge/Safari)
2. اضغط على **"Add to Home Screen"** أو **"تثبيت التطبيق"**
3. سيظهر التطبيق كأيقونة على الشاشة الرئيسية
4. يعمل **بدون إنترنت** بعد التثبيت الأول

### ⚙️ الإعدادات المطلوبة
تأكد من وجود الملفات التالية في مجلد `js/` (غير مرفقة في المشروع الحالي):
- `audio.js` - مدير الصوتيات
- `achievements.js` - نظام الإنجازات
- `splash.js` - شاشة البداية
- `error-handler.js` - معالج الأخطاء
- `adaptive-ai.js` - الذكاء التكيفي
- `store.js` - التخزين المحلي
- `app-version.js` - إدارة الإصدار
- `animations.css` - الرسوم المتحركة

## 📖 دليل الاستخدام | Usage Guide

### 📝 إضافة أسئلة يدوياً
1. افتح **"إدارة الأسئلة"** من القائمة الجانبية
2. اضغط **"✚ سؤال جديد"**
3. أدخل: نص السؤال، الإجابة الصحيحة، 3 إجابات خاطئة
4. اختر التصنيف، الوحدة، الدرس، ودرجة الصعوبة
5. (اختياري) أرفق صورة
6. اضغط **"حفظ"**

### 🤖 استيراد أسئلة من الذكاء الاصطناعي
1. اذهب إلى **"إدارة الأسئلة"** → **"كيفية الاستيراد"**
2. انسخ البرومبت المخصص
3. أرسله لـ ChatGPT/Claude مع ملف PDF للكتاب
4. احفظ النتيجة كملف `.txt`
5. اضغط **"استيراد أسئلة"** واختر الملف

### 📤 تصدير الأسئلة
1. اذهب إلى **"إدارة الأسئلة"**
2. اختر التصنيف من القائمة المنسدلة
3. اضغط **"تصدير أسئلة"**
4. سيتم تحميل ملف JSON يحتوي على أسئلة هذا التصنيف

### 🌐 مشاركة عبر QR
1. اذهب إلى **"إدارة الأسئلة"** → **"مشاركة QR"**
2. اختر التصنيف المطلوب
3. سيتم إنشاء رمز QR يحتوي على الأسئلة
4. امسح الرمز من جهاز آخر لاستيرادها

## 🌐 وضع الشبكة (WebRTC) | Network Mode

### 👨‍🏫 كـ أستاذ (Teacher)
1. افتح **"الإعدادات"** → **"الشبكة"**
2. اضغط **"وضع الأستاذ"**
3. سيتم إنشاء غرفة مع رمز QR
4. شارك الرمز مع الطلاب
5. تحكم بالأذونات (استيراد/حذف) لكل جهاز متصل

### 👨‍🎓 كـ طالب (Student)
1. افتح **"الإعدادات"** → **"الشبكة"**
2. اضغط **"وضع الطالب"**
3. امسح رمز QR الخاص بالأستاذ
4. انتظر قبول الاتصال

> **ملاحظة:** يستخدم التطبيق WebRTC مع STUN/TURN servers للاتصال عبر NAT.
